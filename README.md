# European Football Analytics Lab

Proyecto personal para aprender SQL, Python, Data Analytics y fundamentos de Data Science usando datasets de fútbol europeo.

## Objetivo
Construir habilidades técnicas aplicadas a casos reales de deporte.

## Stack inicial
- GitHub
- GitHub Codespaces
- Python
- SQL
- Jupyter Notebook
- Notion

## Estructura
- datasets/: datos crudos y procesados
- sql/: consultas SQL
- python/: notebooks y scripts
- analysis/: hallazgos e insights
- dashboards/: visualizaciones
- notes/: apuntes de aprendizaje

## Roadmap
- Fase 1: SQL
- Fase 2: Python
- Fase 3: Analytics
- Fase 4: Data Science
